# Библиотека сайта tavda.info

Долгое время эти материалы были частью самого сайта tavda.info. Однако, после нескольких модификаций кода я решил вынести их на отдельный домен, который будет работать быстрее, благодаря статическим страницам, а также у него будет более удобное оформление именно для чтения как электронной книги, чем основной сайт с его перегруженным оформлением.
